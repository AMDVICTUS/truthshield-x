# 🛡️ TruthShield X - AI Fraud Detection Platform

A futuristic, cyber-themed AI-powered fraud detection platform with 8 different fraud detection scanners! Built with Streamlit and Three.js for beautiful 3D effects.

## ✨ Features

### 8 Fraud Detection Modules
- 📱 **Message Scam Detection**
- 📧 **Email Phishing Detection**
- 🔗 **URL Analysis**
- 🖼️ **Deepfake Detection**
- 📋 **Fake Job Scam Detection**
- 💰 **Investment Fraud Detection**
- ❤️ **Romance Scam Detection**
- 💳 **Payment Fraud Detection**

### Cyber UI
- 🎨 Dark cyber theme with neon cyan accents
- ✨ Animated 3D background with particles and rotating shield
- 📊 Analytics dashboard
- ⚙️ Settings page
