# 🛡️ TruthShield X - AI Fraud Detection Platform

A futuristic, cyber-themed AI-powered fraud detection platform with 8 different fraud detection scanners! Built with Streamlit and Three.js for beautiful 3D effects.

## ✨ Features

### 8 Fraud Detection Modules
- 📱 **Message Scam Detection**
- 📧 **Email Phishing Detection**
- 🔗 **URL Analysis**
- 🖼️ **Deepfake Detection**
- 📋 **Fake Job Scam Detection**
- 💰 **Investment Fraud Detection**
- ❤️ **Romance Scam Detection**
- 💳 **Payment Fraud Detection**

### Cyber UI
- 🎨 Dark cyber theme with neon cyan accents
- ✨ Animated 3D background with particles and rotating shield
- 📊 Analytics dashboard
- ⚙️ Settings page

## 🛠️ How to Use Locally

### Installation
1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate venv:
   - Windows: `venv\Scripts\activate`
   - macOS/Linux: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`

### Run
```bash
streamlit run app.py
```

Then open `http://localhost:8501` (or the URL shown in the terminal) in your browser!

## 📦 Deployment to Streamlit Community Cloud

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repo, branch `main`, and file `app.py`
5. Deploy!
